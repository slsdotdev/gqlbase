# @gqlbase/cli

Command-line interface for gqlbase. Provides schema transformation, configuration loading, and file watching.

This is an internal package. Install the main [`gqlbase`](https://www.npmjs.com/package/gqlbase) package instead.

## What this package provides

- `gqlbase` CLI binary
- `defineConfig` — type-safe configuration helper
- `resolveConfig` — configuration resolution from file and CLI options
- Watch mode with debounced file change detection
